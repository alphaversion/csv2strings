# csv2strings

generates Localizable.strings and strings.xml.

```
csv2strings {source path}.csv {output directory path}
```

### example csv file

| id | ja | en | ko | zh-Hans | zh-Han
| -- | -- | -- | -- | -- | --
| # common |   |   |   |   |  
| ok | OK | OK |   |   |  
| cancel | キャンセル | Cancel |   |   |  
| close | 閉じる | Close |   |   |  
| save | 保存 | Save |   |   |  
| done | 完了 | Done |   |   |  
| # settings |   |   |   |   |  
| settings_title | 設定 | Settings | 설정 | 设置 | 設定
